ES6 笔记之 Object

Jason Zhang
2018-02-28
es6
es6

对象类别

  • 普通对象

拥有 JS 对象所有默认行为。

  • 奇异对象

其内部行为在某些方面有别于默认行为。

  • 标准对象

ES6 中定义的对象，比如 Array, Date 。可能是普通的，也可能是奇异的。

  • 内置对象

在脚本开始运行时由运行环境提供的对象。所有的标准对象都是内置对象。

对象字面量语法扩展

  • 属性初始化器速记法

新的方法

  • Object.is(paramA,paramB)

比较两个 参数是否相等，通常情况下运算结果与严格相等运算符 ===一致,例外的两种情况：

  1. Object.is(NaN, NaN) true
  2. Object.is(+0,-0) false

  • Object.assign(reciver, supplier)

    用于组合对象使用，参数个数不限，但是后面的参数会覆盖前面object 的同名属性。

自有属性的枚举顺序

  • 所有数字类型键，按升序排列。
  • 所有字符串类型，按照被添加的顺序排列。
  • 所有符号类型键，也按照被添加的顺序排列。

更强大的原型

  • 修改对象的原型

    直到 ES5, JS 编程最重要的假设之一就是对象的原型在创建以后会保持不变。尽管有 Object.getPrototypeOf() 方法来从指定对象中获取其原型，但是并没有提供修改其原型对象的方法。

    ES6 通过 Object.setPrototypeOf() 方法改变了这种假定，它需要接收两个参数，需要被修改原型的对象，被指定成为前者原型的对象。
