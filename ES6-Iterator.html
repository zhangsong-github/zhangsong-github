迭代器

迭代器是专门设计用于迭代的对象。所有迭代器都有 next 方法，会返回一个结果对象。该对象有两个属性，一个是对应下一个值的 value ，另一个是 布尔类型的 done，表示迭代是否完成（无 value 可以返回）。

生成器

生成器函数表达式，只要在 function 关键字与圆括号之间 使用一个星号即可。
